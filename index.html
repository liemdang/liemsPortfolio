Memory

Dies ist ein Memory Spiel für 2 Spieler. Man wählt für ein Spiel eine Anzahl zwischen 8 und 20 Kartenpaaren aus, die in zufälliger Reihenfolge angeordnet werden. Die Bilder für die Memorykarten werden über eine URL aufgerufen, die eine Bildersammlung für Roboter bereitstellt.

IP Address Tracker

Dieses Projekt ist eine Challenge von Frontendmentor. Hier wird auf die Geolocation API zugegriffen und zeigt den Standort des Hosts einer IP Adresse oder einer Domain an.
Für die Darstellung der Map wurde Leaflet.js verwendet. Die Anwendung wurde nach dem Mobile First Prinzip entwickelt.
